## Misskey Ollama Bot v1.0.0

Initial public release of the project.

### Included
- Misskey mention and reply bot
- Ollama / OpenAI-compatible text backend support
- Docker Compose deployment
- relationship-based access control
- automatic follow-back
- optional ComfyUI image generation
- safer image reply visibility clamping for generated images

### Notes
- ComfyUI is expected to run externally and is not bundled in this repository
- configure your own `.env` from `.env.example`
- keep secrets and real instance values out of Git
